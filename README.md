# Tracker

Tracker Full Stack Project.

## Technologies

- Python: Django (Back End)
- Node: Next (Front End)
- NGINX (Server)
- Docker (Containerization)

## How to start applicaiton?

- Run this command to start the Django, Next and Nginx Servers
  ```bash
  docker-compose up -d
  ```
- Play with the application at http://127.0.0.1
